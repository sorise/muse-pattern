## [C++ 设计模式 备忘录](#)

**可复用的面向对象设计方法**：设计模式(英语 design pattern)是对**面向对象**设计中反复出现的问题的解决方案，每一个模式描述了一个在我们周围不断重复发生的问题以及该问题的解决方案的核心。这样，你就能一次又一次地使用该方案而不必重复劳动。
